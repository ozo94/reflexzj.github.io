python内建函数（序列相关）

发表于 2017-03-30 | 分类于 python语法

python常见的与序列相关的内建函数有4种（sorted()、reversed()、enumerate()、zip()）
sorted()、reversed()是返回的一个列表对象
enumerate()、zip()返回的是一个迭代器

zip()函数

zip([seql, …])接受一系列可迭代对象作为参数，将对象中对应的元素打包成tuple，然后返回由这些tuples组成的list。若传入参数的长度不等，则返回list的长度和参数中长度最短的对象相同。
