python数据读取

发表于 2017-04-01 | 分类于 python语法

参数读取，数据读取，这是两个不同的概念。
参数设置是在程序编译之前，所设置的对应参数
数据读取则是执行在程序编译之后，默认接受用户在控制台通过键盘键入的数据

参数

在pycharm中，通过配置运行环境，预设好参数

通过argv[]获取参数

数据

[raw_input/input]

raw_input([prompt]) 函数从标准输入读取一个行，并返回一个字符串（去掉结尾的换行符）
input([prompt]) 函数和 raw_input([prompt]) 函数基本类似，但是 input 可以接收一个Python表达式作为输入，并将运算结果返回。

[sys.stdin.readline()]

标准输入读取一行，但是会读取换行符’\n’
